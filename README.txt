Для запуска exe файл нужно перейти в каталог exe -> dist -> start -> start.exe


Управление в игре осуществляется клавишами {w, a, s, d} или стрелочками. Выстрел из оружия происходит при нажатии клавиши ЛКМ.
При запуске игры герой "спавнится" на 1 уровне. С этого момента начинает дейвствовать таймер. На карте находятся определенное 
количество противников, которых нужно обязательно победить, иначе вам не откроется портал, для прохождения на следующий уровень.
На карте также разложены аптечки, который восстанавливают жизни героя. При прохождении 3 уровней, вам предложат записать ваше имя
для попадения в статистику игры. Лучших 15 игроков можно посмотреть в разделе "statistics" на главном меню.

Для выхода из игры можно воспользоваться клавишей "escape" или нажать на крестик в верхнем правом углу игры.
